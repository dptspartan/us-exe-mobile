# Spark notification sounds

Bundled for Expo push (`app.json` → `expo-notifications` → `sounds`).

| File | Spark |
|------|--------|
| `buzz.wav` | Two short beeps (noot-style). Replace with your own clip — keep the filename `buzz.wav`. |
| `love_you.wav` | Soft chime |
| `need_hugs.wav` | Gentle two-tone |

After replacing files, rebuild the dev APK (`npm run eas:dev:android`).
